# Traffic Segmentation using SAM and Detectron2

This project focuses on traffic scene segmentation using SAM (Segment Anything Model) and Detectron2. It includes tools for generating COCO annotations and experimenting with model training.

## Files
- `creatingcoco-annotations-from-sam-3.ipynb`: Generates COCO annotations from SAM outputs
- `ep5000-10-lr0-0001-repeat-2.ipynb`: Training and experimentation notebook

## Features
- Traffic image segmentation
- SAM-based mask generation
- COCO annotation creation
- Detectron2-compatible pipeline

## How to Use
1. Open notebooks in Jupyter
2. Run cells step by step
3. Provide traffic image dataset
4. Generate annotations and train models

## Technologies
- SAM (Segment Anything Model)
- Detectron2
- Python, Jupyter Notebook

## Output
- COCO format annotation files
- Segmentation results
